Country Clustering Project

Это проект по unsupervised learning
Модель группирует страны по похожести без правильных ответов

Используется датасет с Kaggle
Country Dataset

Модель делит страны на кластеры на основе экономических и социальных показателей

Используемые признаки
child_mort
exports
health
imports
income
inflation
life_expec
total_fer
gdpp

Используемые технологии
Python
pandas
scikit-learn
joblib
streamlit

Алгоритм
используется K-Means clustering

данные сначала нормализуются через StandardScaler
затем модель разбивает страны на группы
лучшее количество кластеров выбирается через silhouette_score

Структура проекта

train.py
обучает модель
выбирает лучший k
сохраняет модель и scaler

test.py
позволяет вводить данные через терминал
и определяет кластер

app.py
создает простой веб интерфейс
показывает данные
принимает ввод
и выводит кластер

Запуск

обучение модели
python train.py

запуск теста
python test.py

запуск сайта
streamlit run app.py

Установка библиотек

pip install pandas scikit-learn streamlit joblib

Описание кластеров

каждый кластер это группа стран с похожими характеристиками
например
один кластер может содержать страны с низким доходом
другой со средним
третий с высоким уровнем жизни

Результат

модель автоматически группирует страны
и позволяет определить к какой группе относится новая страна
